# 可视化数据转换流水线

## 项目目标

构建一个完全在浏览器运行的数据转换工具。用户导入 CSV、JSON 或 XLSX 文件，通过拖拽节点组成流水线，预览每一步结果，并导出 CSV、JSON 或 XLSX。项目不依赖后端、账号系统或数据库服务。

## 成功标准

- 新用户可以在 5 分钟内完成“导入、清洗、聚合、导出”流程。
- 10 万行、20 列的 CSV 在目标开发机器上可以导入和执行，界面不长时间冻结。
- 同一项目文件重复执行产生一致结果。
- 所有转换都在本地完成，文件内容不发送到网络。
- 核心引擎、文件解析和关键交互具有自动化测试。
- AI 参与过程有真实、可审计的记录。
