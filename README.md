# Study Knowledge Base

这是一个以 Markdown 为基础的个人学习知识库。

## 目录

- `inbox/`：尚未整理的临时笔记
- `topics/`：按主题沉淀的知识
- `resources/`：书籍、文章、课程等资料索引
- `templates/`：常用笔记模板

## 当前项目

- [可视化数据转换流水线](topics/visual-data-pipeline/README.md)：产品范围、架构、开发计划、任务拆分与 AI 使用记录

## 使用方式

1. 新内容先记录到 `inbox/`。
2. 定期整理并移动到 `topics/` 下对应的主题目录。
3. 在笔记之间使用 Markdown 相对链接建立关联。
4. 使用 Git 提交每次有意义的整理和更新。
